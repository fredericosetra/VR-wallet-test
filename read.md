# Projeto

projeto é um teste para uma carteira de cartões digitais.

# Iniciar o projeto.

### Necessário iniciar com 2 comandos:

- 1º comando para iniciar o projeto é:

```
npm run start
```

- 2º comando para iniciar o json-server

```
npm run json
```

# Tecnologias

- React-Native
- Expo
- Styled-Components
- Axios
- Prettier
- Eslint
- json-server
